# Python-text-search-program
Python program that searches a body of text and returns the location of matching terms.

## Features
- Stores a sample body of text
- Prompts the user to enter a search term
- Searches for matching terms within the text
- Prints the location where the term is found
- Allows the user to quit by pressing Enter

## Technologies Used
- Python
